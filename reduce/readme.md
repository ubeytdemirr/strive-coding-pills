# Reduce

## Syntax

**With traditional function**

```js
arr.reduce(function (accumulator, currentValue) {
  // return result from executing something for accumulator or currentValue
}, initialAccumulatorValue);
```

**With arrow function**

```js
arr.reduce((accumulator, currentValue) => {
  // return result from executing something for accumulator or currentValue
}, initialAccumulatorValue);
```
